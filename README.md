# Proto 開發說明

題目：取捨 (Trade-off)
Unity 版本：6000.4.0f1

## 核心設計意圖

以血量作為資源，交換你活下去的每一秒!

倒數結束後依然存活著便是勝利。
每秒、以及衝刺 都會消耗血量，需要謹慎使用衝刺。

---

## 操作

WASD：移動
Space：消耗血量加速

---

## 實作重點

- 實作基本移動、加速、邊界行為  
- 實作血量和加速及加速時扣血
- 建立敵人生成和移動行為

---

## 妥協與遺憾(Scoping)

割捨掉了

- 敵人依波段數隨機設定速度
- 與敵人的戰鬥互動(可以耗血攻擊敵人)
- RougueLike 元素(增加最大血量、減少衝刺需要的扣血)
- 更多樣的敵人(不同體型、移動策略)

---

## 如果延伸會怎麼做

- 移動才掉血，不移動可以維持血量

---
